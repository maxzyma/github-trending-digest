# GitHub Trending 月报 — 2026 年 6 月

_agent/skills 生态大厂化 + vanity 方法论完成收口的一个月；真信号集中在成熟基础设施回流与开源模型/事件驱动发版，风险集中在短龄高失衡的投资/交易/视频类 AI 仓。_

_数据源: https://github.com/trending（daily）｜ 本月 24 个有效 daily analysis（06-02 ~ 06-30）+ 4 份周报（W23~W26）汇总 ｜ 缺漏 6 天：06-01 / 06-04（仅 raw）/ 06-14 / 06-21 / 06-27 / 06-28_

## 月度趋势纵览

6 月没有单一爆点事件，而是四条结构性主线在整月里持续演化，并叠加一条贯穿全月的方法论主线。理解本月 GitHub Trending，先抓这几条线：

**主线一：agent/skills 生态是全月绝对主线，并从「个人 skill 集刷榜」加速演化为「大厂/平台方主导的 agent-ready 基础件」。** "skills" 关键词在 24 个有效日里出现约 18 天。上半月还是个人/社区 skill 集当家（obra/superpowers 全月 6 天在榜、mvanhorn/last30days-skill 7 天、phuryn/pm-skills、addyosmani/agent-skills、mattpocock/skills 140K stars 跨周）；进入下半月，官方入场密集且成为头部锚点——openai/plugins（Codex 官方 180+ 插件示例，6/06 起 5 天）、google/skills（Cloud Next 2026，6/09 首入 +461）、google-labs-code/design.md（6/25 首入 → 6/26 ↑9 蹿至 #1，设计系统 manifest 规范）、anthropics/claude-plugins-official（6/24 首次成为第三方 skills 赛道的官方权威锚点）、aws/agent-toolkit-for-aws、alibaba/page-agent（6/26，HN 147 分，本月 GUI agent 类 grounding 最扎实）。**判断**：agent 工具化在 5 月被推成「事实默认品类」之后，6 月的关键变化是「谁在主导供给」——从草根 KOL/IP 合集转向 Anthropic/Google/OpenAI/AWS/阿里官方项目同月同框，赛道进入平台方收口阶段。但一个悬念整月未解：mattpocock/skills 140K、codebase-memory-mcp 连续千级增量这类超大 star 池，始终缺少可见的第三方深度使用证据，「收藏式 star ≠ 工程采用」的落差贯穿全月。

**主线二：成熟开源基础设施集中回流/首入 tracker，是本月最扎实的真信号。** 这条线从月中逐步强化到月末成为头号真信号主线，且回流主体有清晰迁移：W24（6/08~6/14）以 SaaS 替代品为主（restic/mattermost/chatwoot/opencv/llama.cpp）；W25（6/15~6/21）两轮回潮开源 SaaS 替代品（penpot/insomnia/twenty/RocketChat/plane）；W26（6/22~6/28）回流主体转向数据/文档/抓取基础设施——firecrawl（137K，6/23 首入）、Stirling-PDF（83K，6/23）、opendatalab/MinerU（69,627，6/26）、NanmiCoder/MediaCrawler（52,801，6/26）、IceWhaleTech/CasaOS（34,856，6/26）、smicallef/spiderfoot（14 岁 OSINT 老牌，6/22）同框；月末 06-29/06-30 续上 simplex-chat、free-for-dev、MinerU、VeraCrypt 等重量级老项目。**判断**：这批项目多为 Trending 算法周期性回流而非新发版事件，但它们本身是被广泛依赖的稳态工具，对 infra watcher 与 AI builder 是全月最该优先点开的一类。回流主体从 SaaS 替代品转向数据/文档/抓取基础设施，与 RAG/agent 数据管线的持续需求高度吻合。

**主线三：「去 agent 化」在月中出现又被证伪，说明 agent 存量主体地位并未动摇。** W24（6/08~6/14）出现明显「去 agent 化」——6/10、6/13 两天「真 NEW 全在 agent 赛道之外」，新弹药转向 CV（opencv/supervision）、边缘感知（espectre）、医疗 AI（openmed）、系统容器（apple/container）、推理缓存（LMCache）；但 W25 用一个干净反转证伪了这一「拐点」假说——6/17 agent 本体几乎挂零走到极致，6/18 却被 agent/skills 框架（superpowers 231K、mattpocock/skills 133K、codebase-memory-mcp）集体回潮一天打回。**判断**：6 月中旬的去 agent 化是「成熟项目集体回流挤占头部席位」的表象，而非 agent 供给侧枯竭；头部在「agent 工具」与「成熟基础设施/模型」之间高频切换，是本月稳定的读榜模式。

**主线四：真发版/事件驱动的「双强」项目构成本月技术含金量最高的一批。** 与短龄营销仓相对，本月有一批「真实发版 + HN 头条 + 真采用」三强样本：apple/container（WWDC26 + 1.0 正式版 + HN 1,197 分，6/12 ↑16 登顶）、n0-computer/iroh（1.0 + HN 1,351 分 + Holochain/Fedimint/Start9 真采用，6/17~6/19 跨三天领跑）、zai-org/GLM-5/GLM-5.2（HN 768/883 分 + Artificial Analysis 开源权重榜首 + SWE-bench Pro 超 GPT-5.5 + MIT，6/19）、Lightricks/LTX-2（19B 音视频联合生成、单卡 4090 可跑）、LMCache（被 NVIDIA Dynamo/Red Hat llm-d/字节 AIBrix/vLLM 同时采纳为 KV cache 标准层）。**判断**：开源模型真发版在 W25（6/19）集中注入一次后 W26 即冷却，未延续成稳定主线；但「发布会驱动 ≠ 无机热度」在本月被 apple/container 与 iroh 反复证实——关键看 HN/watchers/真采用是否对得上，不能一刀切按「品牌投放」处理。

**主线五（贯穿全月的方法论线）：本 tracker 的 vanity 判据在 6 月完成收口。** 这是本月最重要的分析方法论进展，逐周递进：W23 明确「高 stars/subscribers 比值作为单一 vanity 指标已失效」并成熟为组合判据（新仓 + 单一作者/组织 + 名人/媒体投放 + 缺第三方验证）；W24 把「高比值」明确降格为**入口信号而非结论**（LMCache 193:1 / openmed 100:1 / espectre 107:1 都高比值却是真采用）；W25 落成**可信度三档分层模板**（真信号 / 需结合足迹 / 收藏型）；W26 进一步细化为**高比值的三种成因**——① 模板复制型（ai-website-cloner「Use this template」天然推高 fork）、② 厂商主导型（hyperframes/page-agent 大厂明星项目传播广、深度订阅少的结构性高比）、③ 真营销驱动型（TREK/ai-berkshire/hiring-agent 短龄 + 高增速 + HN/Reddit 零独立讨论 + 自有渠道投放）。**判断**：全月核心叙事是「传播热度 ≠ 真实采用」，且校准焦点从「是否刷星」转向「真实产品被放大器（KOL/大厂/媒体）加速」——vanity 已从「空壳刷星」演化为「真实产品 + 放大器」，判据必须问「真实使用足迹落在哪个平台」（HuggingFace 下载 / 下游集成 / 论坛 / HN / 第三方客户端），而非只看 GitHub 比值。

## 如何理解热度信号

本月方法论完成收口（详见主线五），核心要点固化为可直接套用的判据：

1. **stars 是注意力，不是采用量。** trending 排名看的是加权/相对增量而非绝对值——成熟老项目今日增量个位/十位数也能在榜（算法周期性回流），OpenMontage 五天暴涨 13K stars 但独立生产案例始终未对齐。高增速首先是「被看见」，不等于「被用起来」。
2. **高 stars/subscribers 比值是入口信号，不是定罪依据，且要区分三种成因**：① 模板复制型（`Use this template` 天然推高 fork，如 ai-website-cloner）；② 厂商主导型（大厂明星项目传播广、深度订阅少的结构性高比，如 hyperframes 366:1、page-agent 337:1）；③ 真营销驱动型（短龄 + 高增速 + HN/Reddit 零独立讨论 + 自有渠道投放，如 TREK 481:1、ai-berkshire 397:1）。同样 337:1+，page-agent 有 HN 147 分托底，与营销驱动短龄仓不可混为一谈。
3. **成熟大仓的高比值是结构性常见。** firecrawl/MinerU/Stirling-PDF/apple-container/turso（242:1）/Pake（205:1）这类长期项目或工具型基础设施，watch 行为本就稀缺，比值结构性偏高，不应单看比值判 vanity。
4. **真正的 vanity 判据是组合**：短龄 + 单一作者/组织 + 名人/公司/KOL 投放 + 缺第三方独立验证 + 足迹缺失。判 vanity 的决定性变量是「真实使用足迹落在哪个平台」——HuggingFace 下载量、下游集成采纳、第三方客户端、HN/论坛/Discord 热度，都是无法靠投放伪造的真信号。
5. **数据口径提示**：本月 daily frontmatter 的 `new_count` 含脚本 NEW 误判（尤其 6/24~6/26 连续三日误标全部为 NEW），真 NEW/RE 以各日正文人工校正后的口径为准，不要直接采信 frontmatter 计数。

## 风险观察

本月需结合社区足迹谨慎解读的高传播样本（保留证据、措辞校准）。按风险成因归类：

- **名人 + 加密币 + benchmark 造假闭环（本月最极端样本）**：MemPalace（6/06 NEW，53,905★ / 2 个月 / 173:1）——演员 Milla Jovovich + 加密 CEO 用 Claude 写码 48 小时登顶 + 自称「100% LongMemEval」被 X Community Note 证伪（实为对 3 道错题 hand-tune）+ 关联加密币 24h pump-and-dump + Kotaku 称「Snake Oil」。把多种刷星手法拼成单一闭环，是 6 月 vanity 演化的标志性节点。**vanity** **compliance**
- **数字仓静默刷星**：affaan-m/ECC（全月 5 天在榜，208K★，198:1，HN/Reddit 持续双零）——「agent harness 性能优化系统」，纯静默刷星典型；career-ops（W23~W24，约 250:1，HN 仅 2 条 0 评，三平台零有机讨论，声量集中作者自营渠道）。**vanity** **bus_factor**
- **AI 投资/交易/决策类短龄高失衡仓（全月最主要 vanity 群体）**：ai-berkshire（397:1）、daily_stock_analysis（跨周在榜，fork:star≈0.9:1 呈 template 复制）、Vibe-Trading（6/29）、council-of-high-intelligence、TREK（481:1）、OmniRoute、video-use 等散布全月。特征一致：比值极高、issue 稀薄、无独立社区讨论、repo 龄短。**vanity** **cost** **compliance**
- **AI 创作/视频高增速仓**：OpenMontage（全月 8 天在榜、跨周霸榜单日增量、五天涨 13K stars，105:1，X 纯模板营销 + 6 周疑停更 + 无外部案例）、palmier-pro（209:1，YC/X 营销 + 单人主导 + 强平台门槛）、hyperframes（366:1，HeyGen 厂商主导）。创作/视频是真实需求方向，但具体项目「star 曲线 ≠ 工作流落地」。**vanity** **bus_factor** **stability**
- **营销跑在工程前面 + 安全隐患**：flowsint（W23，3 个月冲 4.5K★ + 6 安全 advisory 含 critical RCE + LICENSE Apache vs 官网 AGPL 口径矛盾）——「星量暴涨叠加高危漏洞与 license 矛盾」的危险信号。codebase-memory-mcp（全月 7 天在榜，187:1，论文 66 语言/10x 与项目页 158 语言/120x 营销数字不一致，全月无独立第三方深度使用证据）。**vanity** **stability** **bus_factor**
- **抗审查/隐私合规簇**：MasterDnsVPN/fanqiang/iptv-org/Free-TV-IPTV/GhostTrack/OpenWA/SmsForwarder/zapret-discord-youtube 等散布全月——多涉 ToS/版权/隐私边界，需结合使用地法规判断。其中 zapret-discord-youtube（#5704 聚集 720 条真实求助）有真实地缘需求支撑，与营销驱动仓不同。**compliance** **privacy**
- **超大老仓比值误判风险（反向提醒）**：freeCodeCamp（44 万星）、iptv-org（12 万星）、firecrawl（137K）、MinerU、apple-container 等成熟仓高比值属结构性，经多源 grounding 均为真实采用，不应触发 vanity 警惕。
