## 本周趋势纵览

本周是"**Agent 全栈工具化霸榜** × **Vanity 密度创纪录（94%）** × **vanity 判据成熟为组合判据**"三线叠加的"agent 基础设施收割周"，叠加两个本周首次确认的结构性现象（**6/07「虚高在前真货在后」两极分化** + **Agent 安全议题从 flowsint/mxc 两端同时浮现**）。

**主线 1 — Agent 全栈工具化从「六维矩阵」演化为「吃满整张榜」，「给 agent 装手脚/装脑子/省 token」几乎是唯一主题**：W20 识别的"Skill 包六维矩阵"在 W23 进一步扩展——本周榜单每一天的底层主题都不是某个具体产品，而是**围绕编码 agent 的配置层 / 记忆层 / 省 token 层 / UI 层 / 数据接入层 / 沙箱隔离层全栈工具化**。逐日切片：6/02 至少 8/17 直接服务 agent 工具链（hermes-webui agent 前端 / compound-engineering-plugin 官方插件 / revfactory/harness meta-skill / can1357/oh-my-pi 终端 agent / pbakaus/impeccable agent 设计技能 / supermemory agent 记忆层 / fff「为 AI agents 而生」检索 / Scrapling 向 agent skill marketplace 转型）；6/03 头部 headroom + ECC 双双定位"agent harness 性能优化"；6/05 agent 基础设施扎堆 6 项目（headroom token 压缩 / hermes-agent / ECC / copilot-sdk / last30days-skill / spec-kit）；6/06 agent harness（hermes-agent/ECC/flue/openclaw-windows-node）+ 工具接入（Agent-Reach/last30days-skill/openai-plugins）+ UI（CopilotKit）+ 记忆（MemPalace）+ token 工程（headroom）+ 官方 SDK（copilot-sdk）**几乎吃满整张榜**；6/07 头部聚集 last30days-skill / MemPalace / Agent-Reach / CopilotKit / superpowers 全是 agent/skill 仓。**判断**：W20 末提出的"Skill 包已从开源协作产物演化为 AI 时代内容营销载体"在 W23 完成扩张——"agent 配套工具"已是 GitHub Trending 当前**事实上的默认品类**，技术深度全面让位赛道卡位 + 标准窗口期投放。下个观测窗口应关注"agent 工具链是否出现收敛"（赛道过密后头部项目互相替代 / 真实采纳分层）。

**主线 2 — Vanity 密度逐日攀升到本 tracker 纪录（6/06 16/17 = 94%），MemPalace 是「营销造假集大成」样本**：W20 把 vanity 定为"60% 榜单常态"，W23 进一步恶化——逐日 stars:watchers > 100:1 比例：6/02 14/17（82%）→ 6/05 13/14（93%）→ **6/06 16/17（94%，创本 tracker 新高）** → 6/07 14/18（78%）。唯一稳定 organic 反例是 9 岁老仓 jwasham/coding-interview-university（41:1）。本周 vanity 谱系出现三种刷星路径**罕见同框**（6/06 集中）：① 名人金融化——**MemPalace**（NEW，53,905★ / 2 个月 / 173:1）把多种手法拼成闭环：演员 Milla Jovovich + 加密 CEO 用 Claude 写码 48 小时登顶 + 自称"100% LongMemEval"被 X Community Note 证伪（实为对 3 道错题 hand-tune，干活的是 ChromaDB）+ 关联加密币 24h pump-and-dump 创作者分成 50% + Kotaku 直接称"Snake Oil"；② 纯机器静默刷——**affaan-m/ECC**（本周从 204K★ 涨到 208K★，198:1，HN/Reddit 持续双零）；③ 媒体叙事驱动——**666ghj/MiroFish**（64K★，中文媒体"00 后 10 天 vibe coding + 陈天桥 3000 万融资"反复转载）。**判断**：GitHub Trending 算法对"日 stars 增量"敏感、对"watchers 深度参与"无感的结构性缺陷在 W23 被推到极致——榜单应被读为"24h 内组织化推广聚焦榜"而非"自发社区热度榜"，分析重心从"识别 vanity 个例"彻底转向"还原推广路径"。

**主线 3 — vanity 判据本周完成成熟：从「高 ratio 单指标」升级为「组合判据」，6/07 出现「虚高在前、真货在后」结构性两极分化**：这是 W23 最重要的方法论进展。6/02 分析已明确指出"stars:subscribers 高比例作为单一 vanity 指标已失效"——本日 14/17 > 100:1，但 markitdown(138K)/Scrapling(58K)/TradingAgents(81K) 都是活跃多年的真项目，高比例只是 GitHub 早年取消"自动 watch"后的结构性产物。**真正的 vanity 判据已成熟为组合**：新仓 + 单一作者/组织 + 名人或媒体投放 + 缺第三方深度验证。6/07 这一判据迎来最清晰的实证——榜单出现**结构性两极分化**：头部聚集 vanity 膨胀的 agent/skill 仓（last30days-skill 255:1 / MemPalace 173:1 / Agent-Reach 315:1 / superpowers 260:1 219K★），而 **sveltejs/svelte（93:1）、nginx/nginx（32:1）、golang/go（40:1）、openai/whisper（137:1）、vitejs/vite（156:1）五个真有机基础软件老项目首次进入本 tracker，却挤在第 7–15 名、日增 stars 只有 20–30**。注意：whisper/vite/trivy/PaddleOCR 的高 ratio **不是 vanity**——超大热门仓的 watch 行为本就稀缺，ratio 在这里会失真。"虚高在前、真货在后"的同框对照，把"高 ratio ≠ vanity / 低增量 ≠ 不重要"的判据边界一次性讲清。教科书级 vanity 样本本周新增 **santifer/career-ops**（NEW，49K★ / 2 个月仓龄 / 241:1 / HN 仅 2 条 0 评 / Reddit·X 一手帖无法定位 / 外部声量全来自 ProductHunt + 作者自营博客），与 ECC 并列"高 stars + community footprint 严重背离"标本。

**主线 4 — 大厂借 Build 2026 时间窗集中官方投放，「品牌驱动高关注度」与「社区零自发讨论」并存**：本周多个平台厂商背书项目配合 Build 2026 发布节点集中登榜——github/copilot-sdk（6/3 GA，把 Copilot Agent 嵌进任意 app）、openclaw/openclaw-windows-node（Build 2026 微软联合发布、Scott Hanselman 主导）、github/spec-kit（GitHub 官方 spec-driven）、openai/plugins（Codex 插件目录）、NVIDIA/cosmos、以及 6/07 压轴的 microsoft/mxc。它们的共性是**GitHub 一手数据强、官方/KOL 推广触达大，但 HN/Reddit 自发讨论几乎为零**（open-notebook 全靠 XDA/KDnuggets 列表稿、copilot-sdk HN 最高 11 分、cosmos Reddit 零）。与之对照的是**有真实企业实绩的 organic 样本 CopilotKit**（NEW，32K★，刚完成 $27M A 轮，AG-UI Protocol 被 Microsoft Agent Framework / Google A2UI / Amazon / Oracle 采纳、LangGraph/CrewAI/Mastra 首方集成）——两极的差别不在 ratio（都 >100:1），而在"watchers / 企业采纳 / 第三方集成"这类无法靠投放伪造的真实信号是否存在。

**主线 5 — Agent 安全议题本周从两端同时浮现：flowsint「营销跑在工程前面」留下 critical RCE，microsoft/mxc 把 agent 沙箱做成 OS 级强制隔离层**：这是 W23 首次确认的新结构性议题。一端是危险样本——**reconurge/flowsint**（6/03，4.5K★ / 3 个月 / 123:1 / fork:star 12.9% 异常 / 全网零自发讨论）已披露 6 个安全 advisory 含 1 个 critical（命令注入 + Docker 逃逸 → 宿主机 root RCE），且 GitHub LICENSE 标 Apache-2.0、官网却宣称 AGPL-3.0，宣传与实际不符——"星量暴涨叠加高危漏洞与 license 矛盾"是营销跑在工程前面的典型危险信号。另一端是大厂答案——**microsoft/mxc**（6/07，Build 2026 首发 Rust 项目，Microsoft eXecution Container）用声明式 JSON policy 定义 AI agent 能访问的文件/网络/剪贴板/UI，由 Windows 内核运行时**强制**（非事后捕获），启动伙伴含 OpenAI(Codex)/NVIDIA/GitHub Copilot CLI/Manus/Nous Research/OpenClaw。**判断**：当 agent 能"看整个互联网"（Agent-Reach）、能跑任意 skill（superpowers/PAI）时，"怎么把不可信 agent 关进笼子"已成大厂级刚需——agent 工具化的繁荣正在催生其安全对偶。但 mxc 官方明确声明"当前任何 MXC profile 都不应视为安全边界"（early preview），热度来自发布会而非草根验证。
