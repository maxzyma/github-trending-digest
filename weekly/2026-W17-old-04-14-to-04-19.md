本周是 GitHub Trending 的"分水岭周"。**上半周（04-14~16）仍是上周延续**——Karpathy Skills 连续 7 天蝉联榜首、Hermes Agent 维持万级增量、Claude Code 生态项目刷屏。**下半周（04-19~20）发生剧烈换血**——9/10 为新面孔，Karpathy Skills 和 Hermes Agent 同时消失，取而代之的是 Thunderbird AI 客户端、OpenAI Agents SDK、FinceptTerminal 等全新项目。

1. **Claude Code 生态从"通用工具"转向"垂直 Skill"**——上周是 claude-mem、best-practice 等通用增强，本周新增 Android 逆向 Skill、企业架构治理 ArcKit、游戏工作室模板，插件生态正在细分。

## 值得关注的信号

1. **Karpathy 效应有保质期**：andrej-karpathy-skills 连续 7 天霸榜后消失，证明即使是顶级 KOL 的项目，Trending 的自然热度周期也在 7~10 天。项目本身零代码更新（停更于 02-16），纯靠一个 CLAUDE.md 文件跑完全程——这对"什么才是有价值的开源贡献"提出了有趣的问题。

2. **Agent 框架进入公开撕裂期**：Evolver 在 README 首屏指控 Hermes Agent 抄袭其 memory/skill/evolution-asset 设计，并宣布从 GPL 转向 source-available。这是 Agent 赛道从"大家一起做"转向"谁先做的"的标志性事件。OpenAI Agents SDK 同期上榜，巨头入场将加速整合。

3. **Claude Code 插件生态进入垂直化阶段**：上周的 claude-mem、best-practice 是通用工具，本周出现了 Android 逆向 Skill、企业架构治理 ArcKit（68 命令覆盖英国政府规范）、游戏工作室模板（49 Agent）。这条路径与 VS Code Extension 生态早期极为相似——先通用后垂直。

4. **加密代币绑定开源成为模式**：GSD（$GSD）、Voicebox（$VOICEBOX）、Evolver（$EVOMAP）——本周至少 3 个上榜项目带有加密代币。开源+代币的模式正在从偶发变为趋势，但 Stars/Watchers 极端比值（200:1+）暗示这些项目的"star"更多来自投机者而非开发者。

5. **跨界项目值得追踪**：Thunderbird 做 AI 客户端（Thunderbolt）、RuView 用 WiFi 感知人体、Pascal Editor 做浏览器 3D 建筑设计——这些"意想不到的组合"往往比纯 AI 项目更有原创性。Thunderbolt 背后有 Mozilla 品牌，如果执行到位，可能成为开源 AI 客户端的重要选手。
