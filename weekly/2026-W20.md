**主线 3 — Agent 项目首次出现"代码层互引"，trending 项目内部生态拓扑显形**：5/12 同日空降 3 个 NEW 项目代表 agent 生态三新维度（`millionco/react-doctor` 8,217★ AI Coding Agent guardrail / `rohitg00/agentmemory` 4,796★ 跨 harness 记忆层 MCP server / `tinyhumansai/openhuman` 1,525★ 桌面端"个人 AI 超级智能"），W19 已识别 Skill 包"四象限"后，agent 生态横向扩展到 **guardrail + memory + desktop** 三向；5/15 openhuman 仓库 PR #1743（5/14 提交）`feat(memory): optional agentmemory backend (closes #1664)`——**明确把 rohitg00/agentmemory 作为可选 memory 后端集成进 openhuman**，两个本周同时在榜的项目从"并列陈列"升级为"内部依赖"。这是本数字仓观测以来 trending 项目第一次出现 **代码层互引**（区别于以往的"同期榜单巧合"）。判断：**榜单开始具有内部生态拓扑结构**，不再是独立项目快照——如果这是模式，未来 1-2 周可能看到更多 trending 项目互相 import（mattpocock/skills 互相 import 其他 skill 仓库、obra/superpowers 与 K-Dense skills 的代码层兼容等）。**反面警惕**："互捧 stars 经济圈"——openhuman 43 subs / agentmemory 25 subs 都是极低值（vanity 信号强），彼此 PR + 互相在 README 加链接 → 双向引流，真实使用量需进一步验证；后续观测要看是否有**第三方**项目 import 这些 agent 组件（如非 openhuman 的另一 desktop AI 项目 import agentmemory），那才是真实使用扩散信号而非"两人小圈子互证"。

**主线 4 — 首个"榜单冻结日"（5/15）+ 5/17 算法侧波动骤降到 8 项目，trending 算法窗口实证有"远超 24 小时"的滚动期**：5/15 周五本数字仓观测以来**首次出现 15/15 项目与 5/14 100% 同序**——0 NEW、0 RE、0 排名变化、15 项目 `change=`。stars_today 普遍轻微下降但 15 项目仍同时停留在 trending 算法滑窗内有足够速度，无新项目能挤进。**实证**："trending 不是今天最热的 15 个，而是过去 N 小时（远超 24 小时）持续够热的 15 个"——当所有 15 个跨过日 stars 阈值且差距足够大时，新项目就无法插队；冻结实质是过去 1-2 周组织化推广存量项目的稳态，投放窗口外没有新弹药进场。**5/17 反向证据**：榜单从 5/15 的 15 项目骤降到 8 项目，5/14-5/15 在榜的 10 个项目（rohitg00/agentmemory / shiyu-coder/Kronos / roboflow/supervision / influxdata/telegraf / Genymobile/scrcpy / NVIDIA-AI-Blueprints/video-search-and-summarization / CloakHQ/CloakBrowser / mattpocock/skills / github/spec-kit / garrytan/gstack）**集体掉出**——存量推广 momentum 集中放电完后 trending 算法在流量稀薄日只能凑出 8 个跨阈项目（GitHub Trending 算法侧波动）。判断：W20 是观测"trending 算法滚动窗口实际长度"最重要的一周——冻结日 + 骤降日相邻出现说明算法既能在投放高峰时锁定头部一周以上，也会在投放空挡瞬间塌缩到极少项目数；**未来周报应增加"榜单项目数变化曲线"作为算法行为独立信号**。

**主线 5 — 企业生态战 + 100K 里程碑营销正式进入 GitHub Trending 算法层**：5/14 同框 4 个企业组织化推广样本（NVIDIA 720★ 矩阵号驱动 / supertonic HYBE 韩国资本背书 / spec-kit GitHub 官方 + 媒体连发 / gstack YC 总裁个人 IP），叠加 5/13 mattpocock 个人 IP + 课程导流模式，**"AI 时代的开源策略"已不是"做好工具等社区传播"，而是"做工具 + 矩阵号 + PR 稿 + 媒体投放 + 里程碑营销"的组合拳**。5/15 主线三定位的"双 100K 跨阈值倒计时"是该组合拳里程碑营销维度的核心样本——spec-kit 99,540★（距 100K 仅差 460，按 +1,232 速度 1 天必跨，预计 5/16） + gstack 96,820★（距 100K 差 3,180，按 +915 速度约 3-4 天跨过，预计 5/18-5/19）。两个项目跨 100K 之后预期会触发第二轮媒体投放（Twitter Spaces / 博客复盘 / X 头部 KOL 二次推），是 W21 主线观察重点之一。
