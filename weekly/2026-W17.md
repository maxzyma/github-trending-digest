## 值得关注的信号

### 1. Agent 框架进入"诸侯混战"阶段
OpenAI（openai-agents-python）、EvoMap（evolver）、lsdefine（GenericAgent）、Anthropic（Agent SDK）、NousResearch（hermes-agent）——五家 Agent 框架在两周内先后上榜。EvoMap 公开指控 Hermes 抄袭是标志性事件，说明赛道已从"各做各的"进入"直接竞争+知产纠纷"阶段。

### 2. Claude Code 生态从通用工具转向垂直 skill
早期热门是通用型（superpowers、claude-mem、GSD），本周出现 android-reverse-engineering-skill（安全逆向）和 arc-kit（企业架构）——Claude Code 插件生态正在向专业垂直领域深入。

### 3. 安全事故频发暴露 Agent 信任危机
GenericAgent 用户 230GB 文件被误删（#28）、Voicebox 几秒即可克隆声音、Omi 被称 spyware——AI Agent 的"能力越大危害越大"问题在本周集中爆发。无沙盒/无确认的 Agent 正在造成真实伤害。

### 4. 开源项目的 crypto 化趋势加速
GSD（$GSD）、Voicebox（$VOICEBOX）在 README 直接放 Dexscreener 链接，Evolver 绑定 EvoMap 平台。开源 + 代币的模式正在从个别现象变成趋势，侵蚀开源社区信任。

### 5. Mozilla 跨界 AI 的风险与机遇
Thunderbolt 是 Thunderbird 团队从邮件客户端跨界到 AI 的大胆尝试，HN 367 分说明关注度高——但"Mozilla 侧项目历史上经常被废弃"是社区第一反应。MPL-2.0 + 全平台 + 企业级定位是差异化，但面对 Open WebUI/Jan/LibreChat 的先发优势需要证明执行力。
