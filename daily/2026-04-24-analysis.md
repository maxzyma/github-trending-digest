**社区怎么评价**:

### [mksglu/context-mode](https://github.com/mksglu/context-mode) — NEW ⭐ 9,453（+238）| TypeScript | 信源：中偏弱

**它是什么**: 一个 MCP（Model Context Protocol）服务器，专门"**沙箱化**"LLM 调用的 tool 输出——比如 Playwright snapshot 原本 56KB 直接灌给模型，`context-mode` 截获后变成 299 字节的摘要，号称"98% context 节省"。支持 12 个平台（Claude Code / Cursor / Cline / Roo Code / ...）。

**解决什么问题**: LLM agent 调工具时，tool 原始输出（网页 snapshot、GitHub issue list、access log）直接塞 context 窗口，30 分钟后 40% context 就没了。`context-mode` 把原始数据存 SQLite + FTS5 索引，只给模型回召回摘要。

**谁在用、怎么用**（Issues/HN 场景）:
- [HN #47193064](https://news.ycombinator.com/item?id=47193064) 确实存在：**570 points / 107 comments（README 声称的"#1 570+"部分可信，points 数量匹配，但无法独立验证到过 #1）**
- 作者早期 Show HN（#47148025）仅 84 points，README 实际引用的是"第二次更成功的发帖"数据——轻度夸大
- 用户使用场景（作者基准）：Playwright snapshot 56KB→299B、gh issue list 59KB→1.1KB、access log 45KB→155B
- 社区补交 JetBrains Copilot adapter（#327）

**社区怎么评价**:
- 👍 正面: HN 570 points 真实；BetterStack 出技术视频覆盖；2,358 X followers 活跃；第三方 listicle "10 GitHub repos cut Claude Code token usage 60-90%" 纳入
- 👎 负面:
  - **HN 技术质疑（`re5i5tor`）：因 JSON-RPC 协议限制，context-mode 无法真正拦截第三方 MCP 响应，"98% 压缩"只对内建 tools 生效**——核心卖点可能被夸大
  - **#45 作者自挂"Beta testers wanted（12 平台 × 3 OS）"是最热 issue**——说明兼容性早期验证尚未结束
  - #331/#332 better-sqlite3 在 Node v24 SIGSEGV coredump（底层原生依赖稳定性问题）
  - **r/ClaudeAI / r/LocalLLaMA 零自发讨论**；作者在 [issue #134](https://github.com/mksglu/context-mode/issues/134) 公开求助"帮忙传播，我没 marketing 预算"
- 📊 健康度: 创建仅 2 个月即 9.4k stars，现象级增长；contributors 30 人但 **mksglu 独占 590 / bot 191 / 次高仅 36**，bus-factor≈1；近期 commits 多为 `github-actions[bot]` 的 `ci: update install stats` 填充

**风险与争议**:
- **README 的 Microsoft / Google / Meta / Amazon 大厂 logo 无背书证据**——作者只能说"MCP Directory 有 100K+ daily requests"，**未见任一公司公开背书**；这些"员工个人安装 ≠ 企业采用"的 badge 营销法是今日榜单反复出现的模式
- **ELv2 许可证（非 OSI 开源）**：禁止 SaaS 式托管、专利条款严苛，**企业 legal 审查会卡**，不能用于内部 managed service 分发
- **单人 bus-factor=1**：作者自承"无 DevRel、无 marketing，独自维护 + 写文档 + 修 bug"
- **增长真实性**：2 个月 9.4k stars 但 Reddit 零自发讨论，HN 一次爆发 + X 自营销主导——**不排除 launch 集中曝光 + launch 后断崖**

**技术备注**: TypeScript + Bun + SQLite FTS5 + BM25 检索（v1.0.26 起采纳社区建议加入 RRF）；openclaw.plugin.json 插件格式；ELv2 许可。
