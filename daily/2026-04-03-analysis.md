### [siddharthvaddem/openscreen](https://github.com/siddharthvaddem/openscreen) — NEW ⭐ 16K（+2,573）| TypeScript | 信源：中

**它是什么**: 一个免费开源的"录屏美化工具"——录完屏幕后，自动添加缩放、裁剪、标注、变速、背景美化等效果，导出漂亮的产品演示视频。定位是 Screen Studio（$29/月订阅制）的免费替代品。支持 macOS/Windows/Linux，MIT 协议，无水印，商用免费。

**解决什么问题**: 独立开发者和小团队想做产品 demo 视频，但 Screen Studio 要收费，专业剪辑软件学习成本高。OpenScreen 让你零成本制作专业级屏幕演示。

**谁在用、怎么用**:
- 产品 demo 视频制作——独立开发者在社区提 PR 贡献光标跟踪缩放、光标高亮等功能
- Linux/Windows 用户（#157，21 评论）——跨平台渲染极慢是核心痛点

**社区怎么评价**:
- 👍 16K stars，MIT 协议，商用免费——对 Screen Studio 用户有强吸引力
- 👎 **导出管线是最大风险**——视频损坏（#264）、导出卡死（#269）、Windows 导出失败（#256）、解码异常（#245），多个 bug 指向编码/解码链路
- 👎 作者自称"开源新手"，README 明确标注 "very much in beta"
- 👎 HN 仅 1 篇（2 分 / 0 评论），**社区认知度极低**
- 📊 27 contributors（多为零星 PR）；核心维护者 1 人

**风险与争议**: Beta 阶段 + 导出不稳定 + 单人主导。适合关注但不适合投入生产。Star 增长可能受 Trending 短期效应驱动。

**技术备注**: TypeScript + Electron；MIT

### [asgeirtj/system_prompts_leaks](https://github.com/asgeirtj/system_prompts_leaks) — NEW ⭐ 36.4K（+306）| Python | 信源：强

**它是什么**: 一个"AI 产品的内部指令大公开"——收集了 ChatGPT（GPT-5.4）、Claude（Opus 4.6）、Gemini（3.1 Pro）、Grok（4.2）、Perplexity 等主流 AI 产品的系统提示词（system prompt）原文。这些提示词是 AI 产品的"操作手册"——定义了 AI 的行为规则、安全策略、工具使用方式，但用户通常看不到。这个仓库通过各种提取手段将其全部公开。

**解决什么问题**: AI 产品的行为是"黑箱"——你不知道 ChatGPT 为什么拒绝某些请求，Claude 的工具列表长什么样，Gemini 的安全策略怎么写。这个仓库让你直接看到原文，理解 AI 是怎么被"编程"的。

**谁在用、怎么用**:
- Prompt 工程师参考竞品设计——"Claude 的系统提示词超过 24K tokens"成为 HN 热议话题
- 安全研究员分析 jailbreak 攻击面
- AI 产品团队做竞品分析
- 社区自驱动更新——热门 issues 是用户主动贡献新模型 prompt（GPT-5.1、Claude 4.5 Sonnet 等）

**社区怎么评价**:
- 👍 HN **627 分 / 334 评论**（"Claude's system prompt is over 24k tokens with tools"），2025 年度热帖
- 👍 36.4K stars / 6K forks / 495 watchers，该领域最大集合
- 📊 社区自驱动更新机制健康——用户主动提交新模型 prompt

**风险与争议**: **法律灰色地带**——系统提示词的提取可能违反部分平台 ToS。部分 prompt 可能已过期（模型迭代快）。仓库本身无代码，纯 Markdown 归档。

**技术备注**: Markdown 文档集合；定期更新
