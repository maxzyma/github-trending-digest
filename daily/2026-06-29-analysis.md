## NEW 项目分析

### [usestrix/strix](https://github.com/usestrix/strix) — NEW ⭐ 26,770（+122）| Python | 信源：中

**它是什么**: 一个能自己上手做渗透测试的开源 AI agent。你把目标应用交给它，它会像一个安全团队那样分工行动——侦察、找 SQL 注入/XSS/CSRF、写概念验证（PoC）、出报告，最后还能自动提交修复 PR。

**解决什么问题**: 传统漏洞扫描器（DAST）规则死、误报多、不会真正"利用"漏洞。strix 走 validation-first 路线——没跑通 PoC 就不报，目标是把"扫出一堆可疑项让人去核"变成"确认能打通才告诉你"，并用 GitHub Actions 把不安全的 PR 卡在合并前。

**谁在用、怎么用**（来自 Issues/HN）:
- 社区呼声最高的是 MCP 接入（#109，8 条评论），希望把 strix 接进自己的 agent 工具链。
- 工程侧真实痛点：超大 payload 触发 BadRequestError（#460），反映长上下文渗透任务下 LLM 调用稳定性问题。
- 单次扫描 API 成本约 $3-20，用 HTTP proxy + 浏览器 + Kali 容器执行。

**社区怎么评价**:
- 正面: Show HN 102 分 / 15 评论，Help Net Security、freeCodeCamp 等独立媒体有实操覆盖。
- 负面: Reddit（r/netsec、r/cybersecurity）无可索引讨论，offensive 安全社区独立验证仍缺。
- 健康度: stars/subscribers ≈ 209:1，forks 2,987，约 39 contributors，10 月龄仍活跃维护。

**风险与争议**:
- **vanity**: 209:1 比值 + $117M 融资驱动传播，开源版疑为商业版 lead-gen（Datadog 模式）。
- **cost**: 强依赖 GPT-5/Claude/Gemini，无离线模式，深扫 token 成本不可控。
- **compliance**: "零误报"是设计意图非独立基准，无 SOC2/PCI 认证；offensive 工具天然滥用风险。
- **dependency**: 拉取数 GB Kali 镜像，多 agent 并发吃 CPU/内存。

**技术备注**: Python；多 agent 架构；HTTP proxy + headless 浏览器 + Kali 容器沙箱执行。

### [Robbyant/lingbot-map](https://github.com/Robbyant/lingbot-map) — NEW ⭐ 8,253（+372）| Python | 信源：中

**它是什么**: 蚂蚁 Robbyant 具身智能团队的"流式 3D 重建"基础模型。给它一段单目摄像头视频流，它能"边走边看"实时把场景重建成 3D 并定位自身位置，约 20 FPS、可处理万帧级长序列——本质是用前馈神经网络做 SLAM 的事。

**解决什么问题**: 传统 SLAM 依赖几何优化、参数难调、长序列易漂移。lingbot-map 用 feed-forward 基础模型一把出结果，目标是给机器人/自动驾驶/AR 提供更省事的实时空间感知。

**谁在用、怎么用**（来自 Issues/arXiv）:
- 研究者在认真跑基准：催 benchmark eval 脚本（#70）、问 7-Scenes 复现设置（#68）。
- 技术质疑扎实：闭环轨迹存在 loop closure 漂移（#60，14 评论讨论最热）、重建点云分层（#27）、追问是否做全局位姿图优化（#78，暗示当前无闭环优化）。

**社区怎么评价**:
- 正面: 有 arXiv 论文、ETH3D/7-Scenes/Oxford Spires 基准复现，蚂蚁背书。
- 负面: HN 仅 45 分 / 3 评论热度浅；X 以官方新闻稿分发为主；Reddit 无讨论。
- 健康度: stars/subscribers ≈ 103:1，57 open issues，805 forks；LinZhuoChen 71 commits 绝对主导。

**风险与争议**:
- **bus_factor**: 单一核心贡献者主导。
- **stability**: 闭环漂移、点云分层等重建质量问题待解，缺全局位姿图优化。
- **vanity**: 103:1 比值 + HN/X/Reddit 独立讨论稀薄，外部传播靠官方公关；但有真实 issue 技术讨论支撑，属轻度。

**技术备注**: Python；feed-forward 3D foundation model；单目 RGB 流式输入，对标 SLAM。

- **stability**: 4 月创建，实盘连接器自称实验性。

**技术备注**: Python；多券商/交易所连接器；策略导出 Pine/TDX/MQL5。

**技术备注**: Swift；macOS 原生；本地推理 + 闭源 Fluid-1 增强层。
