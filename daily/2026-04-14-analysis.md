### [jamiepine/voicebox](https://github.com/jamiepine/voicebox) — NEW ⭐ 16,385（+512）| TypeScript（Tauri/Rust） | 信源：中

**它是什么**: 一个本地运行的"语音克隆工作室"——你给它几秒钟的声音样本，它就能克隆出那个人的声音，然后用克隆的声音说任何你想说的话。支持 23 种语言，5 个 TTS 引擎（Qwen3-TTS、LuxTTS、Chatterbox 等），全程本地运行，声音数据不上传。自称"开源版 ElevenLabs"。

**解决什么问题**: ElevenLabs 订阅贵（$11-99/月）、必须上传声音到云端、有隐私顾虑。Voicebox 完全本地运行——免费、无隐私泄露、不受订阅限制。对于个人创作者、播客制作者、无障碍需求来说是强需求。

**谁在用、怎么用**:
- 播客/有声书制作：多声音时间线编辑器（类 DAW）
- 个人声音克隆：30 秒音频即可克隆（Medium 评测"genuinely impressed"）
- 开发者集成：内置 REST API，可作为语音后端嵌入其他应用
- #185（32 评论，最热）：Fine-tune 教程需求——用户想进一步优化克隆效果
- #141/#84/#8（共 72 评论）：**GPU 不可用是头号痛点**——默认 CPU 推理极慢

**社区怎么评价**:
- 👍 X/Twitter 传播力强——Hasan Toor 推文获大量转发，作者 Jamie Pine 定位精准 "gonna make this the ollama of voice"
- 👍 r/LocalLLaMA 帖子"97ms 延迟语音克隆"引发关注
- 👍 Tauri/Rust 构建，原生性能（非 Electron）；MIT 协议
- 👎 **单人项目**——jamiepine 473 commits 占 97%+，第二贡献者仅 7 commits
- 👎 **停更 2 周**（最后推送 03-31）；213 open issues 无人处理
- 👎 **$VOICEBOX 加密代币**已出现——crypto 投机信号
- 👎 HN 仅 1 帖 4 分 0 评论——HN 社区几乎无认知
- 📊 16,385 stars / 115 watchers（142:1）；1,947 forks；213 open issues；核心 1 人

**风险与争议**: Jamie Pine 的上一个项目是 Spacedrive（也有 $SPACEDRIVE 代币），模式类似——高调开源 + 社交媒体营销 + crypto 代币。Voicebox 功能确实实用，但单人维护 + 213 open issues + 停更 + 代币 = 可持续性存疑。更深层的伦理问题：几秒音频就能克隆声音，语音认证体系面临根本性挑战。

**技术备注**: TypeScript（Tauri/Rust）；MIT；5 TTS 引擎（Qwen3-TTS 为主）；23 语言；REST API；macOS + Windows，Linux 未发布

### [gsd-build/get-shit-done](https://github.com/gsd-build/get-shit-done) — NEW ⭐ 52,192（+655）| JavaScript | 信源：强

**它是什么**: 一个 Claude Code 的"大脑增强系统"——给 Claude Code 装上项目管理能力：自动研究 → 拆解任务 → 子 agent 并行执行 → 自动提交代码。核心卖点是"解决 context rot"（上下文窗口填满后 AI 质量退化）。通过 meta-prompting + spec-driven 开发模式，让 Claude Code 在长会话中保持输出质量。`npx get-shit-done-cc@latest` 一键安装。

**解决什么问题**: Claude Code 在长对话中会"变笨"——context window 填满后，后续输出质量明显下降。GSD 通过将大任务拆成原子化 spec、用 subagent 隔离上下文来缓解这个问题。但同时引入了大量 meta-prompting 开销——trade-off 是"用更多 token 换更好的结构"。

**谁在用、怎么用**:
- 全栈项目开发：定义 PRD → 自动研究 → 原子任务拆分 → subagent 并行执行
- 多项目管理：#1948 要求 handoff 带项目标识（多项目用户的真实需求）
- 有用户声称一个月用 GSD 生成 25 万行代码（HN，无信源验证）
- #1888（10 评论）：**Windows 上 bash 命令要 30 分钟**——性能 Blocker
- #803（35 评论，最热 closed issue）：**讨论阶段被自动跳过**——AI 自己回答了该问用户的问题

**社区怎么评价**:
- 👍 HN 讨论热度极高（473 points / 254 comments），评价两极分化
- 👍 结构化 prompt + plan-review agent 确实能提前发现问题（多个 HN 用户确认）
- 👎 **"highly overengineered"**——多个 HN 用户认为直接用 Claude 就能达到类似效果，GSD 多烧了一堆 token
- 👎 **"endless planning loops that waste tokens"**——有用户反映 planning 循环不收敛
- 👎 HN 讽刺："适合网红一夜搭个 Potemkin SaaS，Superpowers 才出实际成果"
- 👎 **$GSD 加密代币在 Dexscreener 上线**——README 直接放 Dexscreener 徽章，开源项目发币是重大红旗
- 📊 52,192 stars / 225 watchers（232:1，极高）；4,372 forks；30 open issues；核心 2 人（glittercowboy 915、trek-e 488 commits）

**风险与争议**: README 直接内嵌 **$GSD 代币 Dexscreener 链接**——这在开源开发工具中极为罕见，将开源信誉与 crypto 投机绑定。Stars/Watchers 232:1 说明"收藏多、持续关注少"。4 个月 5.2 万星的增速与 Hermes Agent（同赛道）类似，存在社交媒体营销驱动增长的特征。12000 行 CJS 单体正在向 TypeScript SDK 重构（#2007/#2008），架构转型期风险高。与 BMAD、Speckit、Ralph、Superpowers 等同类工具竞争激烈。

**技术备注**: JavaScript/TypeScript；MIT；npm 包 `get-shit-done-cc`；支持 Claude Code/OpenCode/Gemini CLI/Kilo/Codex 等多个 AI IDE；context engineering + meta-prompting
