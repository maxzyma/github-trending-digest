**社区怎么评价**:
- 👍 存活近 4 年（2022 年创建），10K stars，社区互动深入
- 👎 HN 完全不感兴趣（2 次出现均 0 评论）——"内容农场工具"不符合 HN 审美
- 📊 GPL-3.0；Reddit API 政策收紧是持续风险

**风险与争议**: Reddit API 收紧可能随时影响抓取。本质是"内容农场基础设施"，平台政策风险高。长期看 AI 视频生成工具会取代截图拼接方案。
